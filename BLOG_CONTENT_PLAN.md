## Content Pillars

### Pillar 1: Buyer's Guides (High Commercial Intent)

#### Post 1: "How to Choose the Right Board Game Table Size"
- **Target keywords**: "board game table size", "gaming table dimensions"
- **Slug**: `/blog/board-game-table-size-guide`
- **Outline**:
  - Room size calculations (table + 3ft clearance per side)
  - Player count considerations (4 vs 6-8)
  - Game type requirements (sprawling wargames vs card games)
  - Standard door clearance for delivery
  - *Tie-in: Standard vs Grand comparison*

#### Post 2: "5 Features Every Board Game Table Needs"
- **Target keywords**: "board game table features", "what to look for gaming table"
- **Slug**: `/blog/board-game-table-features`
- **Outline**:
  - Recessed vault/pit (why 10cm depth matters)
  - Convertible dining top (dual-purpose value)
  - Rail/accessory system
  - Quality playing surface
  - Durable finish for spills
  - *Myth-bust: Cup holders, LED lighting (nice-to-have, not essential)*

#### Post 3: "Board Game Table vs Regular Gaming Table"
- **Target keywords**: "board game table vs gaming table", comparison queries
- **Slug**: `/blog/board-game-table-vs-gaming-table`
- **Outline**:
  - Define both categories
  - Key differences (vault depth, surface material, accessories)
  - Who each serves best
  - Price point expectations

#### Post 5: "DIY Board Game Table vs Buying: The Real Cost"

  - Common mistakes and fixes

#### Post 6: "Kickstarter Board Game Tables: Backer's Guide"
